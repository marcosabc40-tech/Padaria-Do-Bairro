README - Padaria do Bairro

Descrição do Projeto
--------------------
Este projeto consiste em uma página web simples em HTML que apresenta a "Padaria do Bairro".
O objetivo é praticar a estruturação de páginas HTML utilizando elementos semânticos,
tabelas, imagens, vídeos e iframes.

A página apresenta informações institucionais, produtos, galeria de imagens, vídeos,
horário de funcionamento e dados de contato da padaria.

Tecnologias Utilizadas
----------------------
- HTML5
- Imagens (JPEG/PNG)
- Vídeo local (MP4)
- Vídeo incorporado do YouTube
- Google Maps (iframe)

Estrutura do Projeto
--------------------
/index.html        -> Página principal do site
/videos/
  padaria.mp4      -> Vídeo institucional da padaria
/imagens/ (opcional)
  fotos da padaria -> Caso deseje armazenar imagens localmente
README.txt         -> Documento explicativo do projeto

Principais Seções da Página
---------------------------
- Cabeçalho com nome e apresentação da padaria
- Seção "Sobre Nós" com a história da padaria
- Lista de produtos e diferenciais
- Tabela com produtos, descrição e preços
- Galeria de imagens em formato de tabela
- Vídeo institucional e vídeo do YouTube
- Horário de funcionamento
- Contato com mapa, endereço, telefone e e-mail

Como Executar o Projeto
-----------------------
1. Faça o download ou copie todos os arquivos do projeto.
2. Certifique-se de que o arquivo index.html está no diretório principal.
3. Clique duas vezes no arquivo index.html ou abra-o em qualquer navegador moderno
   (Google Chrome, Firefox, Edge, etc.).

Observações
-----------
- Os preços e informações são fictícios e utilizados apenas para fins educacionais.
- As imagens externas são carregadas a partir de URLs públicas.
- O projeto não utiliza CSS externo, podendo ser estilizado futuramente.

Autor

Projeto desenvolvido para fins educacionais e prática de Programação Web.
